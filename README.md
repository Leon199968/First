# 极速触点 · 射击游戏

一款基于HTML5 Canvas的射击类反应速度游戏，使用专业瞄准镜精准射击彩色移动目标！

## 🎮 游戏特色

### 🎯 射击体验
- **专业瞄准镜** - 红色发光瞄准镜，具有刻度标记和十字线
- **精准射击** - 鼠标实时跟踪，瞄准镜跟随移动
- **爆破效果** - 击中目标产生华丽的粒子爆炸效果
- **物理模拟** - 粒子具有重力、阻力和生命周期

### 🏆 游戏模式
- **多难度选择** - 入门、简单、普通、困难、专家五个难度等级
- **动态目标** - 不同颜色的圆形目标在场景中随机移动
- **边界反弹** - 目标碰到边界时会反弹，增加挑战性
- **实时统计** - 分数、时间、连击、命中率实时显示

### 🔧 游戏功能
- **暂停系统** - 按ESC键随时暂停/继续游戏
- **音效控制** - 可开启/关闭游戏音效
- **成绩记录** - 自动保存各难度的最佳成绩
- **响应式设计** - 支持不同屏幕尺寸

## 🚀 快速开始

### 在线游玩
访问 GitHub Pages 链接直接游玩：
```
https://您的用户名.github.io/speed-touch-game/
```

### 本地运行
1. 克隆仓库
```bash
git clone https://github.com/您的用户名/speed-touch-game.git
cd speed-touch-game
```

2. 启动本地服务器
```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js
npx http-server

# 使用 Live Server (VS Code 扩展)
```

3. 打开浏览器访问
```
http://localhost:8000
```

## 🎯 游戏玩法

1. **选择难度** - 点击画面选择适合的难度等级
2. **开始射击** - 使用瞄准镜瞄准移动的彩色目标
3. **击中目标** - 点击鼠标射击，击中目标获得分数
4. **连击奖励** - 连续击中目标可获得连击奖励
5. **暂停游戏** - 随时按ESC键暂停或继续游戏

### 🏅 难度等级

| 难度 | 目标数量 | 移动速度 | 目标大小 | 颜色数量 |
|------|----------|----------|----------|----------|
| 入门 | 3个 | 慢速 | 大 | 2种 |
| 简单 | 4个 | 慢速 | 较大 | 3种 |
| 普通 | 5个 | 正常 | 中等 | 4种 |
| 困难 | 6个 | 快速 | 较小 | 5种 |
| 专家 | 8个 | 极速 | 小 | 6种 |
